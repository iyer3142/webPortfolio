Projects

Things I have worked on...

  • Custom LoRaWAN (IN865) Gateway Development using RAK2245 & Raspberry pi

    Developed an in-house gateway for the purpose of improvising LoRaWAN coverage for smart-city projects, that controlled smart streetlights. Deployed in-premises Application server & Network server along with postgres DB servers to manage data frames from the gateway. Configured IS865 LoRa band as we localisation standards.

  • Access Control HRM Device with contactless thermal screening

    Designed electronics circuitry and modelled 3D drawing for plastic injection-moulding to build an unbeatable product by leading a team of professionals. Showcasted project management abilities, design-thinking and improvising UI and UX.

  • Home Automation using Particle Photon and custom webApp Implementation

    Designed a system to monitor and control domestic electrical appliances wirelessly over the internet using a web application, to automate processes like water tank management. Developed a fully in-house web-application using HTML, CSS and Javascript and made electrical connections with a relay board.

  • Building a refrigerator using Peltier devices

    Constructed a mini-fridge using thermo-electric Peltier devices. Temperature control was achieved using a thermostat.

  • Ambient Temperature Monitor using Arduino and DHT11

    Display the ambient room/outdoor temperature and humidity.

  • Automated Outdoor Lighting based on illuminance

  • Motion Sensing Lights using Arduino and Passive-IR (PIR) device

    Automatically turn on/off lighting depending upon natural light.
