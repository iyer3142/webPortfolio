Projects

DALI-2 Driver Integration
Developed a master controller for DALI-based LED drivers for indoor and streetlighting automation projects. DALI stands for Digital Addressable Lighting Interface. It is a 2-way communications protocol that is used to provide control over, and communication between, the components in a lighting systems. Using one controller several LED drivers can be controlled on the same line. DALI uses manchester-coded signals running on 12-24V DC line.
4G Modem
Interfaced 4G modemfor a Remote Terminal Unit product used in smart street Lighting Feeder Panels. With the slow phasing out of 2G and 3G cellular networks and also to improve speed and reliability of internet backhaul, it was time to switch to 4G.
MODBUS RTU Multi-function Energy Meter Communication
Dived deep into the world of MODBUS over RS485 and learnt the beautiful language of HEX values to parse energy parameters from the meter like Voltage, Current, PF, Freq, kWh, KVar, MDI, etc. Challenge was the conversion of HEX to Floating numbers to implement a high voltage and high current cut-off feature to save around 200 streetlights from getting damaged. Implemented this for a 3-Phase IoT Power Feeder Panel featuring 40 Ampere load capability on each phase.
LoRaWAN Gateway
Developed a LoRaWAN gateway using Raspberry Pi and RAK gateway concentrator for smart-city projects that controlled LoRa end-nodes in smart streetlights. Deployed in-premises Application server & Network server to manage data frames from the gateway.
Schedule Timer over Bluetooth
Designed hardware and firmware for a bluetooth-based schedule timer to turn ON/OFF output of high-load feeder panels based on either static ON/OFF times or an astrologically calculated values of sunrise and sunset time that automatically functions as per timezone and geography, also compensating for seasonal changes.
